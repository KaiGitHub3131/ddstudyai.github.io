# 叮当AI学习笔记

这是一个现代化的AI学习笔记分享网站，记录AI学习过程中的心得体会、技术笔记和实践案例，分享AI知识与经验。
## 功能特点

- 响应式设计，适配各种设备屏幕
- 现代化UI界面，采用蓝色主题
- 文档分类展示和搜索功能
- 精选文档和最新更新展示
- 技术周刊订阅功能
- 平滑滚动和动画效果

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Font Awesome 图标
- AOS 动画库

## 如何使用

直接打开 `index.html` 文件即可在浏览器中查看网站。

## 自定义

您可以通过修改以下内容来自定义网站：

1. 修改 `tailwind.config` 中的颜色配置来更改主题色
2. 更新文档卡片内容和图片
3. 添加或删除技术分类
4. 调整响应式布局断点

## 注意事项

- 网站使用了CDN加载外部资源，请确保网络连接正常
- 图片资源使用了远程链接，实际部署时建议替换为本地资源
